# WebRTC  Server

## Clone Repository

Clone the repository to your local environment.

```sh
git clone https://github.com/KhaledElKenawy00/webrtc_server
```

### Server Setup

#### Step 1: Go to  webrtc_server folder

```js

cd webrtc_server

```

#### Step 2: Install Dependency

```js
npm install
```

#### Step 3: Run the project

```js
npm run start
```
